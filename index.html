Admin Portal

Dashboard

In side the admin panel there some basic overwiew of the whole system. The sidebar with the menu item to navigate to different pages, the maintenance message notification, calender, and a chart to show the transaction overview. It also has a card design overview of the current asset, tenant, owner, occupied asset and total income count

The owerview are dynamic, so based on the current data it will be updated.

Assets

In the left side menu bar you will see a list of different pages and after the Dashboard option comes the Assets option.

The Assets has two sub-level menu,

  • Add New assets
  • Assets List
Using this two sub-level menu you can add new assets to the system and see the list of the asset with other major information of the assets. Plus when you add assets you automatically creates some other data and relations for the system and a account under assets to keep track of the rent and maintenance charges.

Add New Asset

Add New Assets: If you click to the add new assets button you will be directed to a form page where you will find a form to add the new assets to the system.

There are Property Address, Type of Assets, No. of Bed, No. of Bath, Square feet, Value of Property, Net Rent Value, Issues With the Property, Deposit Amount form field that needed to be filled. Please note that you do not need to fillup the following input field DH Charges, Tax, Gross Rent as they are calculated automatically based on the type of assets you selected earlier. As for different types of assets, the charges and the tax varies so this will be automated function in this form. To set those charges and tax we have also created another page. We will explane it later in the documentation.

On the right side you will find an option to upload an assets image. So press button and you will be promt to select an image from you computer, select the image and you will see a preview of the image over the default image. Note that to add an asset into this system you have complete and fillup all the input fields in this form or else you will not be able to add the assets in the portal.

After suceessfully completing the form fillup with the image click button below to submit the form. If everything is ok, your assets will be added to the portal system.

Assets List

Asset List is where you will see all the assets listed in the portal.

Here the assets are listed in a table with some important information of the assets. If you look closely you will see that these are the information you entered when listing the asset inside the portal. Apart from those information another three column are also visible. They are Owner ID, Tenant ID, Due Payment.

The Owner ID is the id of the owner assigned with this assets. So after you add an asset you need to add owner details of the assets as well, in that section you will assign assets under an owner. Adding an owner will be described later. If any asset has no owner assigned, you will also notice it here by a label Owner Not Assigned Yet

The Tenant ID is the id of the tenant who is currently occupying the asset. If there is no tenant in the asset, you will see a notification label in the row stating Vacant – No Tenant Yet

Due Payment is the column where you will see the dues of the assets, the rent or maintenance payment might be due for that specific asset. So you will see that which asset has what amount to money due.

The Details button is used to see all the information in details. The will see the owner details the tenant details and the due details of that asset

As this details will help to create an owner portal account. Where the owner will be able to acces and see all the assets of them registered under DiamondHead.

After you sucessfully add an owner in the portal, you will be able to see the list of the owner.

In this page you will again find a form to fillup. Where you need to add the owner details.

On the right side you will see an option to add image for the owner. Which is also a required option. So you need to add image of the owner.

After creating a owner user you need to save them and pass the credential to the owners. They need these credential to access the portal to their own. It means that when you click on this field you will see all the list of assets which are not assgned any owner. Those assets which has its assigned owner, you will not be able to see them in this list.

Do not get confuse if you can not see the other assets which you know have no owner assigned. we are just showing only 5 items here. Other are also available here. When you click on a specific asset there will be check mark/icon on the right side of the assets name. Which indicated that you selected that assets

You can select multiple asset using this method, all you have to do is select the asset.

To remove a check mark or deselect the asset you have to click on them again. It works like a checkbox present in any other website. Based on you select you are assigning the owner with the assets. So in a way you can assign multiple assets under one Owner

Payment Overview

The payment overview page is where you will find the overview of all the payment paid

In this page there are two tabs,

  1. Rent Payment
  2. Maintemance Payment

Rent Payment
Rent payment tab is where you will see all the rent payment made by the tenant. The Payment Date, Asset ID, tenant ID, Payment ID and the Amount.
The button will be to show the asset details and all the transaction details of the payment made by the tenant.
When there is no payment complete, then you will see the No Rent Payment Complete Yet label. There is also search filter option on the top right of the table. where you will be able to filter and search using date, asset id, tenant id and even payment id.

Maintenance Payment
Maintenance payment tab is where you will see all the maintenance payment made by the DiamondHead. The Payment Date, owner ID, asset ID, maintenance description and the Amount. As DiamondHead will pay for the maintenance and send the bill or invoice to the owner. DiamondHead has the option to send this message with the detils to the owner and deduct the amount of maintenance from the rent which will owner receive.
The button will be to show the asset details and all the transaction details of the payment made by the DiamondHead.
When there is not payment then you will see the No Maintenance Payment Complete Yet label. There is also search filter option on the top right of the table. where you will be able to filter and search using date, asset id, owner id.

Payment Due

The payment due page is where you will find the dues or all the payment that will be paid

You will also find a total payment due amount on top. These total amount is all the payment you will receive from the owner and tenant as maintenance cost and rent.

In this page there are two tabs,

  1. Rent Payment
  2. Maintemance Payment

Rent Payment
Rent payment tab is where you will see all the rent payment will be made by the tenant. The Payment Date, Asset ID, tenant ID, Payment ID and the Amount.
The button will be to show the asset details and all the transaction details of the payment that will be made by the tenant.
When there is no payment due then you will see the No Rent Due label. There is also search filter option on the top right of the table. where you will be able to filter and search using date, asset id, tenant id and even payment id.

Maintenance Payment
Maintenance payment tab is where you will see all the maintenance payment will be made to the DiamondHead. The Payment Date, owner ID, asset ID, maintenance description and the Amount. As DiamondHead will pay for the maintenance and send the bill or invoice to the owner. When the owner approved the maintenance then DiamondHead has the option to deduct the amount of maintenance from the owner's rent.
The button will be to show the asset details and all the transaction details of the payment made by the DiamondHead.
When there is not payment then you will see the No Maintenance Due label. There is also search filter option on the top right of the table. where you will be able to filter and search using date, asset id, owner id.

Maintenance Request

The Maintenance Request is a form where you will be able to send maintenece request to the DiamondHead

DiamondHead will receive the form and take action based on that request.

You have to fillup the form and at the end click on the button to send the request

Message Board

The Maintenance Message Board is where you will see all the maintenece request made to the DiamondHead

DiamondHead will receive them but in your portal you will see that the message has a differnt color background. It will change its background color to green only when the Diamondhead Management has viewed your request on their end. Its like a notification that your maintenance request has been seen by the authority.

You have to refresh the page using the refresh button on top of the message table. You will also be able to see the number of message/request you have sent.
